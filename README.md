# ?? Portafolio de Proyectos en C#

Esta carpeta contiene proyectos desarrollados en **C#**, utilizando tecnologías como **.NET**, **Windows Forms**, **ASP.NET Core**, y **Unity**. Están orientados al desarrollo de aplicaciones de consola, aplicaciones gráficas, servicios web y videojuegos.

---

## ?? Lista de proyectos

### 1. ?? `console-calculator`
Aplicación de consola que simula una calculadora básica con operaciones matemáticas.

- Tecnología: .NET 6 (Console App)
- Características: entrada por teclado, bucles, condicionales, manejo de errores

### 2. ??? `windows-forms-notes`
Aplicación de escritorio para tomar notas simples usando Windows Forms.

- Tecnología: .NET Framework o .NET 6 (WinForms)
- Características: interfaz gráfica, guardar y cargar notas, manejo de archivos

### 3. ?? `aspnet-api`
API REST básica creada con ASP.NET Core para la gestión de tareas.

- Tecnología: ASP.NET Core 6
- Características: CRUD, rutas con controllers, almacenamiento en memoria o base de datos

### 4. ?? `unity-platformer`
Juego 2D de tipo "plataformas" desarrollado en Unity.

- Tecnología: Unity Engine + C#
- Características: físicas 2D, animaciones, controles, escenas múltiples
